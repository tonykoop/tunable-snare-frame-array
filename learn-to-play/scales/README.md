# Scales & Patterns — Tunable Snare Frame Array

> **Stub** — patterns pending prototype build.

## Available pitch set

```
D3 (Drum A) — A3 (Drum B) — D4 (Drum C)
```

This is a power-chord voicing (D–A–D): root → fifth → octave.
It fits rock/metal feels, modal drones, and ostinato percussion lines.

## Basic pulse patterns ★☆☆

**Single-drum quarter pulse** — pick one drum, tap center on every beat:
```
Beat:  1  2  3  4
Drum:  A  A  A  A
```

**Three-drum rotation** — cycle through all three:
```
Beat:  1  2  3
Drum:  A  B  C  A  B  C ...
```

## Two-drum cross ★★☆

**Bass + treble split** — A on beats 1 & 3, C on 2 & 4, B fills:
```
Beat:  1  +  2  +  3  +  4  +
Drum:  A     C     A     C
           B           B
```

## Snare interplay ★★☆

Enable snare on Drum B only. Play the three-drum rotation above —
Drum B will have snare texture while A and C are clean membrane.
Gradually engage snare on A or C for increasing density.

## Three-drum polyrhythm ★★★

**3 against 2 (hemiola)** — Drum A in 3, Drums B+C in 2:
```
Beat grid (6 units):
A:  ●  .  .  ●  .  .   (every 3)
B:  ●  .  .  ●  .  .
C:  ●  .  .  .  ●  .   (every 2, offset)
```
