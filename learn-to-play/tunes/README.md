## Notation convention (pending)

Tunes will be notated as ASCII grid tablature:
```
Beat:   1  e  +  a  2  e  +  a  3  e  +  a  4  e  +  a
DrumA:  x           x           x           x
DrumB:     x           x           x           x
DrumC:        x           x           x           x
```

Where `x` = center tap, `o` = edge tap, `X` = slap, `-` = rest.

Snare state indicated above each bar: `[snare-A:off | snare-B:on | snare-C:off]`

Add individual tune files here as the instrument is prototyped and played.
