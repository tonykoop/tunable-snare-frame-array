# Assembly Manual — Tunable Snare Frame Array

## Tools for Assembly

- Drum key (standard 8-32 or M5 version matching lug hardware)
- Chromatic tuner (phone app or clip-on)
- Combination square
- M5 hex driver

---

## Part Check Before Assembly

- [ ] 3× welded steel shells (⌀254, ⌀203, ⌀152 mm)
- [ ] 1× plasma-cut mounting frame with flanges bent
- [ ] 3× Remo Ambassador heads (10″, 8″, 6″)
- [ ] 3× counter-hoops (matching each head size)
- [ ] 6+6+4 = 16× tension rods + washers (drum-standard 8-32 or M5)
- [ ] 6+6+4 = 16× drum lugs (bought, mounted to shell lug ears)
- [ ] 3× snare wire mechanisms with thumb-wheel (bought)
- [ ] 12× M5 bolts + nuts for shell-to-frame mounting
- [ ] 3× snare wire bundles (12, 10, 8 strand)

---

## Assembly Sequence

### Step 1 — Mount Shells to Frame (5 min)

1. Insert Shell A (254 mm, largest) into the leftmost frame cut-out from the drum-face side.
2. Pass M5 bolts through frame lug ear holes into shells. Finger-tight.
3. Verify shell sits flush with frame face; no rocking. Tighten M5 bolts to snug.
4. Repeat for Shell B (center), Shell C (right).
5. Check all three shells are parallel (level along top rims).

*[Photo placeholder: frame with 3 shells mounted, rear view showing bolt pattern]*

### Step 2 — Install Lugs on Shell Ears (10 min)

1. If lugs not pre-mounted: bolt each lug to its shell ear using M5 (or 8-32) screw. Lugs should spin freely and not wobble.
2. Inspect all 16 lug anchors — no cross-threading.

*[Photo placeholder: close-up of lug ear + lug on Shell A]*

### Step 3 — Seat Drum Heads (5 min)

1. Place Remo Ambassador 10″ on Shell A. Center it; the head rim should seat smoothly on the bearing edge.
2. Place counter-hoop over head.
3. Thread tension rods through counter-hoop into lugs — finger-tight only. Install all rods before tensioning.
4. Repeat for Shell B (8″) and Shell C (6″).

*[Photo placeholder: head seated on Shell C, counter-hoop in place, rods finger-tight]*

### Step 4 — Install Snare Mechanisms (5 min)

1. For each drum: loop snare wires across the batter head (bottom face of head). The thumb-wheel bracket should be positioned at 6 o'clock (bottom of drum face).
2. Mount bracket to nearest lug ear or dedicated snare bracket weld point.
3. Thread snare wire anchor to bracket; snug thumb-wheel to light contact tension.

*[Photo placeholder: snare wire across head, thumb-wheel at 6 o'clock position]*

### Step 5 — Tune to Target Pitches (20 min)

See `tuning-notes.md` for full procedure. Summary:

1. Cross-tension all rods ¼-turn increments until head rings when tapped.
2. Tap near each lug; equalize pitch around the rim.
3. Strike center; read chromatic tuner:
   - Shell C target: D4 ≈ 294 Hz
   - Shell B target: A3 ≈ 220 Hz
   - Shell A target: D3 ≈ 147 Hz
4. Once within ±50 cents: detune, re-tension 2× more to seat head.
5. Final tune: mark rod extension with paint pen.

*[Photo placeholder: tuner showing D4 on Shell C, rod markings visible]*

### Step 6 — Snare Adjustment and Final Check (5 min)

1. Disengage all snare mechanisms (fully loosen thumb-wheels).
2. Strike each drum; confirm clear tone, no rattle.
3. Re-engage snares halfway; confirm buzz activates on each drum.
4. Play all three drums in sequence: check no resonance transfer between drums.

---

## Maintenance Notes

- **Re-tuning**: temperature and humidity change head tension. Re-check pitch before each play session.
- **Head replacement**: standard Remo Ambassador heads; available at any drum shop.
- **Snare wire**: replace if worn or buzzing unevenly; standard snare wire bundles fit.
- **Frame**: no maintenance required; clean with dry cloth.
